# ModernTeleport
## 简介
一个简介、高效、现代化的传送插件
功能包含：
- /tpa 系列指令
  - go <目标玩家>
  - come <目标玩家>
  - accept <请求玩家>
  - refuse <请求玩家>
  - cancel <目标玩家>
  - auto <目标玩家> \[accept|refuse]
  - ignore <目标玩家>
- /back
- /home 系列指令
